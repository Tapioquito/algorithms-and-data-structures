                    Sorting Algorithms

1) What is 'sorting'?
    - Sorting is the process of rearranging the items in a collection 
    so that the items are in some kind of order.
Examples:
- Sorting numbers from smallest to largest;
- Sort movies based on release year;

2) Why do we need to learn this?
    - Sorting is an incredibly common task, so it's good to know how it works;
    - There are many ways to sort things and different techniques 
    have their own advantaged and disadvantages;

3) Objectives
    - Implement bubble sort
    - Implement selection sort
    - Implement insertion sort
    - Understand why is important to learn these simpler sorting algorithms
